# Skills-Exchange-Frontend

A high-performance, mobile-first Responsive Frontend built with Next.js 14 (App Router), Tailwind CSS, and Lucide React icons for a "Skill Exchange Barter Platform" targeted at the Bangladeshi student market.

## Features
- **Premium UI Overhaul**: Features animated Hero Section, glassmorphism, floating navigation bars, and fully responsive layouts.
- **Skill Post Feed**: Discover and browse skills visually with tailored UI cards.
- **Create Post Modal**: A slick, multi-step progress UI to barter your knowledge.
- **User Profile Dashboard**: Easily manage your posts and respond to trade requests securely.
- **Micro-Animations**: Uses `framer-motion` for layout transitions and gradient blob ambiances.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to freely exchange skills!
